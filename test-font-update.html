Benefits of Inter + Cubano Normal pairing:

  • Consistency: Both are sans-serif fonts with clean lines
  • Readability: Inter is highly legible at all sizes
  • Modern feel: Both fonts have contemporary, clean aesthetics
  • Versatility: Inter works well for body text, Cubano for accents
  • Performance: Reduced font loading with fewer font families
Note: The site now uses a consistent sans-serif typography system with Inter as the primary font and Cubano Normal for special accents and headings where appropriate.
